# Trout Lake Booking Management System

A comprehensive booking and property management system for Trout Lake accommodations.

## Features
- Multi-building room management with visual floor plans
- Booking calendar and client management
- Budget tracking and expense management
- Employee time tracking and task management
- Automated email reminders and invoicing
- Real-time activity monitoring

## Deployment
This application is deployed on Vercel with automatic deployments from the main branch.
